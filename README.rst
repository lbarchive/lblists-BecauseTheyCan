=================
Because They Can!
=================

*Why?*

Open source projects make you ask that question, or gasp

* *Wait, what?*,
* *OMG!*,
* *That's insane!*, etc.

They can be awesome or terribly horrendous ideas to do whatever they are trying
to do.

This list is published under CC0.

Terminal Games
